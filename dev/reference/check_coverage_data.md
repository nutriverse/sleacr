# Check coverage data for post-stratification weighted estimation

Check coverage data for post-stratification weighted estimation

## Usage

``` r
check_coverage_data(cov_df)
```

## Arguments

- cov_df:

  A [`data.frame()`](https://rdrr.io/r/base/data.frame.html) of
  stratified coverage survey data.

## Value

A message or an error on whether `cov_df` is structured appropriately
for post-stratification weighted estimation.
