# Check population data for post-stratification weighted estimation

Check population data for post-stratification weighted estimation

## Usage

``` r
check_pop_data(pop_df)
```

## Arguments

- pop_df:

  A [`data.frame()`](https://rdrr.io/r/base/data.frame.html) of
  population data.

## Value

A message or an error on whether `pop_df` is structured appropriately
for post-stratification weighted estimation.
